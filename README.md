> Expand variables in reflinks.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save expand-reflinks
```

Expands this:

```
[abc]: [bar]/abc/[foo]
[fez]: [qux]/fez
[qux]: [baz]/qux
[baz]: [bar]/baz
[bar]: [foo]/bar [foo]: 123
```

To this:

```
[abc]: 123/bar/abc/123
[fez]: 123/bar/baz/qux/fez
[qux]: 123/bar/baz/qux
[baz]: 123/bar/baz
[bar]: 123/bar\n
[foo]: 123
```

## Usage

```js
var expand = require('expand-reflinks');
console.log(expand('[bar]: [foo]/bar\n[foo]: 123'));
```

Results in:

```
[bar]: 123/bar
[foo]: 123
```
